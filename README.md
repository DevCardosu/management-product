# Sistema de Controle de Estoque

Projeto simples desenvolvido em Java com o objetivo de praticar lógica de programação e conceitos básicos de Programação Orientada a Objetos (POO).

O sistema funciona via terminal e permite o controle básico de produtos em estoque.

## Funcionalidades
- Cadastrar produto (nome, preço e quantidade em estoque)
- Listar produtos cadastrados
- Adicionar quantidade ao estoque
- Remover quantidade do estoque
- Validação básica de quantidade disponível

## Conceitos aplicados
- Programação Orientada a Objetos (POO)
- Encapsulamento
- Entrada de dados com Scanner

## Tecnologias utilizadas
- Java
- Git e GitHub
